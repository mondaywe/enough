### Obtaining Zoho CRM Client ID and Secret

* Open [Zoho API Console](https://api-console.zoho.com/) and sign into your account, or create a new one.
* From within the API console page, go to `Applications`.
* Click `ADD CLIENT` button top right and select `Server-based Applications`.
* Fill in any information you want in the `Client Details` tab
* Go to tab `Client Secret` tab.
* Now copy the Client ID and Client Secret.  
* Set the Redirect URL for OAuth `https://app.revert.dev/oauth-callback/zohocrm`. 


#### Connect to Zoho CRM via Revert

* Create an account on Revert if you don't already have one. (https://app.revert.dev/sign-up)
* Login to your revert dashboard (https://app.revert.dev/sign-in) and click on `Customize your apps` - `Zoho CRM`
