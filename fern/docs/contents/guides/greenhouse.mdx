#### Obtaining Greenhouse API key

-   Open a [Greenhouse account](https://www.greenhouse.com/).
-   Create a new Greenhouse API key, using the steps mentioned [here](https://support.greenhouse.io/hc/en-us/articles/115000521723-Manage-Harvest-API-key-permissions)

#### Connect to Greenhouse via Revert

-   Create an account on Revert if you don't already have one. (https://app.revert.dev/sign-up)
-   Login to your revert dashboard (https://app.revert.dev/sign-in) and click on `Create App` - `Greenhouse`
